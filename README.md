# 🌦 Weather App (React + OpenWeather API)

A simple and responsive weather application built using React.  
It allows users to search for any city and view real-time weather details.

---

## 🚀 Features

- 🔍 Search weather by city name
- 🌡 Displays temperature, humidity, min & max values
- 🌥 Shows weather condition (cloudy, rainy, etc.)
- 🎨 Dynamic UI with weather-based images & icons
- ⚡ Error handling for invalid city inputs
- 📱 Clean and responsive design using Material UI

---

## 🛠 Tech Stack

- React (Vite)
- Material UI
- Formik (Form handling)
- Yup (Validation)
- OpenWeather API
